# 🚀 LibreTV 快速启动指南

欢迎使用增强版LibreTV！这里为您提供快速上手指南。

## 🎯 立即体验新功能

### 📱 1. 安装PWA应用
```
1. 访问LibreTV网站
2. 浏览器会自动提示安装PWA
3. 点击"安装"或"添加到主屏幕"
4. 享受原生应用体验！
```

### 🎨 2. 切换主题
```
快捷键: Ctrl+Shift+T
或 点击左上角主题切换按钮
选择喜欢的8种主题之一
```

### 🔍 3. 智能搜索
```
快捷键: Alt+S
使用语音搜索功能
查看智能建议和热门搜索
```

### ⚙️ 4. 个性化设置
```
点击右上角设置按钮
在个性化面板中：
- 调整播放偏好
- 管理收藏夹
- 查看观看历史
- 设置书签
```

## 🔧 故障排除

### ❌ 常见问题

**Q: PWA安装失败？**
A: 检查浏览器支持，尝试手动添加到主屏幕

**Q: 主题切换不生效？**
A: 清除浏览器缓存，刷新页面重试

**Q: 语音搜索不工作？**
A: 检查麦克风权限，允许浏览器访问麦克风

**Q: 播放卡顿？**
A: 检查网络连接，尝试降低画质设置

**Q: 快捷键不响应？**
A: 确保焦点在页面内，检查是否与浏览器快捷键冲突

### 🛠️ 重置设置
如果遇到问题，可以在设置面板中：
1. 点击"重置个性化数据"
2. 清除所有缓存
3. 恢复默认设置
